4chan Image Scraper
===================

Requirements
------------
Python 3.6+, Requests and BeautifulSoup

Usage Examples
--------------
Single thread link: 

	python3 4chan_image_scraper.py link https://boards.4channel.org/c/thread/3988954

Batch file:

	python3 4chan_image_scraper.py batch path/to/batch,json
